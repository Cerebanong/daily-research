# Daily Email Format

## Subject Line
Daily Research Briefing — {YYYY-MM-DD} ({Day of Week})

## Structure

### Opening
One-sentence overview of the day's most notable finding across all topics.

### Databricks
List each finding as a bullet with:
- **Bold headline** — 2–3 sentence summary. (Source: [Source Name](URL), Date)

If no findings: "No notable Databricks developments today."

### AI Innovation
Same format as above.

### AI Governance
Same format as above.

### Data Governance
Same format as above.

### Closing
"Sources checked: {count} direct sources, {count} web searches across {count} topics."

## Tone
Concise, informational, scannable. Written for a data and AI professional. No marketing language.

## Format
HTML email. Use headers, bold text, and hyperlinks. Keep total length under 1,000 words.
